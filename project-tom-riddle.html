Diary of Tom Riddle — Memory & Persona AI System

Concept

Inspired by the Diary of Tom Riddle, this system explores persistent persona behaviour and long-horizon memory in AI systems.

System Design

  • Handwritten user input capture
  • Handwriting-style output rendering
  • Persona-constrained language generation (Gemini)
  • Stateful memory across interactions

Outcome

Demonstrates controlled AI personality, memory continuity, and separation between generation engines and behaviour rules.
